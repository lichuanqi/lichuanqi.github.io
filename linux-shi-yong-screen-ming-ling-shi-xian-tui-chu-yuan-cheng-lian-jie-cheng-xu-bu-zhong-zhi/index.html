Linux使用screen命令实现退出远程连接程序不终止

· 2022-10-21 ·

1 介绍

screen命令类似于“虚拟桌面”的概念，可以启动一个Linux下的虚拟桌面，在其中可以和普通终端一样执行各种命令。可以实现远程操作服务器把程序运行起来，退出远程后程序不终止

2 安装

# 检查是否安装
screen -ls
# 安装
sudo apt install screen

3 使用

# 创建窗口
screen -S name
# 退出当前窗口
ctrl +a d
# 查看已创建窗口
screen -ls
# 关闭窗口
ctrl+d

# 进入名为name的窗口
screen -r name
# 远程连接进入screen窗口后突然掉线，重连遇到进不去窗口时可以通过以下命令踢掉上一个进入窗口的进程，然后进入窗口
screen -D -r name
